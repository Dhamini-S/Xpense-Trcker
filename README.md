# 💰 Xpense Tracker

A modern, clean, and intuitive expense tracking web application with a premium UI and delightful user experience.

![Xpense Tracker](./screenshot.png)

## ✨ Features

### Core Functionality
- **Demo Login Screen** - Secure authentication with demo credentials
- **Comprehensive Dashboard** - Get a complete overview of your finances at a glance
- **Expense Management** - Add, edit, and delete expenses with ease
- **Smart Filtering** - Filter by category, date range, and amount
- **Expense Notes** - Add detailed comments and notes to each expense
- **Budget Tracking** - Set monthly budgets with visual progress indicators
- **Interactive Charts** - Beautiful visualizations for spending insights
- **Category Breakdown** - Track spending across 7 predefined categories

### Expense Categories
- 🍔 Food
- ✈️ Travel
- 🛍️ Shopping
- 📄 Bills
- 🎮 Entertainment
- ⚕️ Health
- 📦 Others

### Design Highlights
- Modern card-based layout
- Smooth animations and transitions
- Delightful micro-interactions
- Responsive design (desktop & mobile)
- Clean typography and spacing
- Gradient accents and hover effects
- Professional color palette

### Installation

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. No build tools or dependencies required!

### Running the App

Simply open the `index.html` file in your browser:

```bash
# Using Python (if installed)
python -m http.server 8000

# Using Node.js (if installed)
npx serve

# Or just double-click index.html
```

Then navigate to `http://localhost:8000` in your browser.

## 🎯 Key Features Explained

### Dashboard Overview
- **Total Expenses** - Lifetime spending summary
- **Monthly Spending** - Current month's expenses
- **Budget Progress** - Visual indicator of budget utilization
- **Transaction Count** - Total number of expense entries

### Expense Management
- Add new expenses with description, amount, category, date, and notes
- Edit existing expenses
- Delete unwanted entries
- Real-time updates across all views

### Filtering & Sorting
- Filter by category (Food, Travel, Shopping, etc.)
- Filter by date range (Today, Week, Month, Year)
- Sort by date or amount
- Search by description or notes

### Budget Tracking
- Set monthly budget targets
- Visual progress bar with color coding
- Automatic calculation of remaining budget
- Category-wise spending breakdown

### Insights & Analytics
- Spending trend charts (7/30/90 day views)
- Category distribution pie chart
- Top spending categories
- Average daily spending patterns

## 💾 Data Storage

All data is stored locally in your browser's localStorage, including:
- User session information
- Expense records
- Budget settings

**Note:** Data persists across sessions but is tied to your browser. Clearing browser data will reset the app.
