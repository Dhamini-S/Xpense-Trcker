# 🚀 Quick Start Guide - Xpense Tracker

## Method 1: Using the Start Script (Recommended)

```bash
cd xpense-tracker
./start.sh
```

Then open your browser to: **http://localhost:8000**

## Method 2: Direct File Open

Simply double-click `index.html` to open it in your default browser.

## Method 3: Custom Server

If you have Node.js installed:

```bash
cd xpense-tracker
npx serve
```

Or use any other local development server.

## ⚡ Quick Tour

   - Use the demo credentials above
   - Click "Sign In"

### 2. **Dashboard**
   - View total expenses and monthly spending

### 3. **Add Your First Expense**
   - Click "+ Add Expense" button (top right)
   - Fill in details:
     - Description (e.g., "Coffee at Starbucks")
     - Amount (e.g., 5.50)

## 💾 Data Persistence

- All your data is saved in browser localStorage
- Data persists across page refreshes
- Each browser has its own separate data
- Clear browser data will reset the app

---

## 🔄 Reset App

To start fresh with sample data:

1. Open browser DevTools (F12)
2. Go to Console tab
3. Type: `Storage.clearAll()` and press Enter
4. Refresh the page
